Documentation Overview

To edit this theme, basic knowledge about HTML, CSS, JS & Bootstrap is required. You can learn HTML, CSS and JS from W3Schools. You can learn Bootstrap by visiting official site.

Note: Before you make any changes, take a backup of all files and folders.

File Structure

Theme consists of 3 main folders namely css, img & js.
Inside css folder you will find styles used in this theme. style.css is the main stylesheet.
Inside img folder you will find all images used in the theme.
Inside js folder you will find all Javascript files. custom.js is the main js file.

Editing

To edit files you need any code editor. I recommend using Notepad++.

Add Custom Styles

To add custom styles, create new style sheet in css folder. Include the style file before the closing tag of <head>. Make sure that you are including this new file after style.css

If you want to add any extra CSS files, copy the file to css folder and link it before style.css.

<link href="css/FILE-NAME.css" rel="stylesheet">
Add Custom JavaScript

To add JS, just create a new java script file in js folder. Include the newly created js file before the closing tag of <body>. Make sure that you are including this new file after custom.js. Alternatively you can also include Javascript code directly to the html file by enclosing the code in <script> ...your code... </script> and adding this above the closing tag of <body>

If you want to add any extra JS files, copy the file to js folder and link it before custom.js.

<script src="js/FILE-NAME.js"></script>
Editing HTML

Almost all html codes in this theme are neatly commented. All you need to do is to read the comment and modify the content based on your need.

Fonts

This theme uses Font Awesome Icons for Icons and Google Web Fonts for Fonts.

Icon Font

Refer Font Awesome Icons for list of fonts. To use any fonts in the theme, you need to use the following code.

/* Latest version */
<i class="fa fa-XXXX"></i>
Web Font

This theme uses Dosis, Open Sans Condensed, Open Sans. To add or change font visit Google Web Fonts, Choose the font, Copy the @import code. Paste it in top of your stylesheet.

@import url(http://fonts.googleapis.com/css?family=Dosis:200,400,600);
@import url(http://fonts.googleapis.com/css?family=Open+Sans+Condensed:300,700);
@import url(http://fonts.googleapis.com/css?family=Open+Sans:400italic,400,600);

Add New Page

This theme comes with 0-base.html file. This is the base file using which you can create you own page. Create a copy of the page, rename it and open in Text editor. Edit the title in <title> section. Change heading and add your content. Done.

Add Heading 
<div class="entry">
	<!-- Page title -->
	<h2>Page Title</h2>
	<!-- Title Below Content -->
</div>
Add Content 
<div class="entry">
	<!-- Page title -->
	<h2>Page Title</h2>
	<!-- Title Below Content -->
	
	<!-- Add Your page Content Here... -->
	
</div>

Logo

Below code shows the markup of Logo area. You can also use images for logo. Just play with HTML. I reommend using PNG images for logo.

<div class="logo">
	<!-- Logo with font awesome icon. Replace it with your logo. Replace # with your site link -->
	<h1><a href="#"><i class="fa fa-XXXX"></i> LOGO_TITLE</a></h1>
</div>

Navigation

Below code shows the markup of Navigation bar.

<div class="navi">
	<!-- Navigation Starts Here -->
	<div id="ddtopmenubar" class="mattblackmenu">
		<!-- Main navigation -->
		<ul>
			<li><a href="#" rel="ddsubmenu1">LINK1</a></li>
			<li><a href="#" rel="ddsubmenu1">LINK2</a></li>
			<!-- Add rel="ddsubmenu{count}" to main menu -->
			<li>
				<a href="#" rel="ddsubmenu1">MAIN_LINK</a>
				<!-- Sub navigation, A -->
				<!-- Add id "ddsubmenu{count} and class "ddsubmenustyle" -->
				<ul id="ddsubmenu1" class="ddsubmenustyle">
					<li><a href="#">SUB_LINK1</a></li>
					<li><a href="#">SUB_LINK2</a></li>
				</ul>
			</li>
		</ul>
	</div>
</div>
